# MasMenu

### Demo
> [See here]

### Installation
Add library to your project from dist folder:
```sh
<script src="scripts/masmenu.min.js"></script>
```

### Usage
```sh
  new MasMenu(options)
```
### Options
```sh
{
  speedPerLetterMs: <Number; Default: 100;>
  randomMode: <Boolean; Default: false;>
}
```
